正则表达式笔记 😑

修饰符

概念

修饰符 含义 描述
i ignore - 不区分大小写 即A 和 a 没有区别
g global - 全局匹配 查找所有的匹配项

举例

    let fruits = "apple,APPLE,banana Banana,apple"
    fruits.match(/apple/) //["apple"]
    fruits.match(/apple/g) //["apple","apple"]
    fruits.match(/apple/gi) //["apple", "APPLE", "apple"]

元字符

概念

元字符 描述
^ 匹配输入字符串的开始位置
$ 匹配输入字符串的结束位置
* 匹配前面的子表达式零次或多次
+ 匹配前面的子表达式一次或多次
[a-z] 匹配指定范围内的任意字符
[^a-z] 匹配任何不在指定范围内的任意字符
\d 匹配一个数字字符 等价于 [0-9]
\D 匹配一个非数字字符 等价于 [^0-9]
\s 匹配任何空白字符，包括空格、制表符、换页符等等。等价于 [ \f\n\r\t\v]
\S 匹配任何非空白字符。等价于 [^ \f\n\r\t\v]
\w 匹配字母、数字、下划线。等价于[A-Za-z0-9_]
\W 匹配非字母、数字、下划线。等价于 [^A-Za-z0-9_]

举例

1.元字符 +

    let string = "123sdsad23ASD19saj086~!@#$hf2136^1hfdg3&*()_+/88"
    string.match(/^[0-9]/) //["1"]
    string.match(/[0-9]/) //["1"]
    string.match(/[0-9]+/) //["123"]
    string.match(/[0-9]/g) //["1", "2", "3", "2", "3", "1", "9", "0", "8", "6", "2", "1", "3", "6", "1", "3", "8", "8"]
    string.match(/[0-9]+/g) //[ "123", "23", "19", "086", "2136", "1", "3", "88" ]

常用的正则表达式

1.去除两边空格

  name = name.replace(/^\s|\s$/,'')
1

2.去除所有空格

  name = name.replace(/\s/g,'')
1

3.只能输入数字

  name = name.replace(/\D/g,'')
  //或者
  name = name.replace(/[^0-9]/g,'')
1
2
3

4.只能输入英文

  name = name.replace(/[^a-zA-Z]/g,'')
1

5.只能输入英文和数字

  name = name.replace(/[^a-zA-Z0-9]/g,'')
1

6.只能输入数字且不能以0开头

  name = name.replace(/[^\d]/g, '').replace(/^0{1,}/g,'')
1

7.验证邮箱

  let mail = '961211746@qq.com'
  let limitMail = new RegExp(/^([a-zA-Z]|[0-9])(\w|\-)+@[a-zA-Z0-9]+\.([a-zA-Z]{2,4})$/) 
  limitMail.test(mail) //true
